Raccourcis clavier Linux

Se déplacer sur le bureau

Alt + F2 Faire apparaître la fenêtre des commandes
Alt + F4 Fermer la fenêtre actuelle
Alt + F10 Maximiser une fenêtre
Alt + F7 Déplacer la fenêtre
Alt + F8 Redimensionner la fenêtre
Alt + Tab Changer rapidement de fenêtre
Super + S Activer le sélecteur d'espace de travail
Super + W Activer le mode "Expo". Affiche toutes les fenêtre de l'espace de travail
Ctrl + Alt + Flèches Basculer entre les espaces de travail
Ctrl + Alt + Shift + Flèches Déplacer la fenêtre actuelle vers un espace de travail différent
Ctrl + Alt + Suppr Se déconnecter
Ctrl + Super + D Cache toutes les fenêtres et affiche le bureau
Ctrl + Alt + L Vérouiller l'écran
